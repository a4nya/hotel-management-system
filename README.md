# Hotel Management System

A Python + MySQL based hotel management system using PyMySQL and a local database.

## Features
- Room management
- Customer booking
- Billing system

## Tech Stack
- Python
- MySQL
- PyMySQL

## How to Run
1. Install Python
2. Install PyMySQL:
   pip install pymysql
3. Create the database using `schema.sql`
4. Run:
   python main.py
